## 概念
### loader
定位：负责加载布隆索引

### builder
定位：负责构建布隆索引

### container
定位：负责存储布隆索引

### controller
定位：负责加载相关配置，控制布隆索引加载、构建整个流程

### cleaner
定位：负责清理本地布隆所文件
